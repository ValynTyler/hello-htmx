# Hello, HTMX!
Quick introduction to HTMX using GO, following an [internet tutorial course](https://www.youtube.com/watch?v=x7v6SNIgJpE)
## Usage
To start the application locally, run `air` in a nix development shell:
```bash
cd <path/to/project>
nix develop .
air
```
